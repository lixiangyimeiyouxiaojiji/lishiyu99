一、项目部署流程
    1. 安装依赖包：npm install
    2. 启动项目：npm run dev
    
二、基本操作
    1. 下载项目 git clone https://git.coding.net/jyx_js/IPRP.git
    2. 更新代码 在项目根目录下执行：git pull
    3. 提交代码
        a. git status 查看变化的代码
        b. git add --all 添加所有的修改文件
        c. git commit -m "备注"
        d. git push
